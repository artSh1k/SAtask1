## Конкурентный анализ

При разработке требований учтён функционал следующих систем-конкурентов:

- **Ostrovok B2B** — моментальное подтверждение, широкая база отелей, API для туроператоров
- **Bronevik** — работа с квотными загрузками и стоп-сейлами, сложные тарифные сетки
- **AcademOnline** — неттинг-расчёты, многоуровневые комиссионные вознаграждения
- **Booking.com for Business** — удобный UX поиска, фильтрация и управление бронями
