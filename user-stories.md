## Роль: Менеджер по бронированию

### Этап: Поиск и сравнение

| ID | User Story | Приёмка (Acceptance Criteria) |
|:---|:---|:---|
| US-01 | Как Менеджер, я хочу искать отели по направлению, датам и числу гостей, чтобы видеть актуальные доступные варианты | При вводе параметров система возвращает список отелей, доступных в заданные даты, с ценой и количеством свободных номеров |
| US-02 | Как Менеджер, я хочу фильтровать выдачу по цене, звёздности и типу питания, чтобы быстро находить подходящие варианты | Фильтры применяются мгновенно, без перезагрузки страницы. Доступен множественный выбор (например, 3 и 4 звезды одновременно) |
| US-03 | Как Менеджер, я хочу видеть итоговую цену с учётом моей наценки и комиссии прямо в поисковой выдаче, чтобы не тратить время на ручные расчёты | В карточке каждого отеля отображается: «Цена нетто», «Наценка», «Итого для клиента» |

### Этап: Бронирование

| ID | User Story | Приёмка (Acceptance Criteria) |
|:---|:---|:---|
| US-04 | Как Менеджер, я хочу создавать предварительную бронь (холдировать номер) нажатием одной кнопки, чтобы зафиксировать цену на время общения с клиентом | После нажатия кнопки «Забронировать» система отправляет запрос поставщику. Номер блокируется на 20 минут (настраиваемый параметр) |
| US-05 | Как Менеджер, я хочу получать уведомления об изменении статуса брони и о приближении дедлайна оплаты, чтобы не пропустить отмену отелем | Уведомления отображаются в интерфейсе (индикатор в шапке) и дублируются на email. Предупреждение за 1 час до истечения тайм-лимита |

### Этап: Документооборот

| ID | User Story | Приёмка (Acceptance Criteria) |
|:---|:---|:---|
| US-06 | Как Менеджер, я хочу автоматически генерировать Ваучер для клиента из данных подтверждённой брони, чтобы избежать ошибок ручного ввода | Кнопка «Скачать ваучер» формирует PDF с данными: ФИО туриста, название отеля, даты заезда/выезда, номер брони, контакты принимающей стороны |

## Роль: Супервайзер / Финансовый отдел

| ID | User Story | Приёмка (Acceptance Criteria) |
|:---|:---|:---|
| US-07 | Как Супервайзер, я хочу видеть отчёт по всем неподтверждённым броням старше 1 часа, чтобы контролировать работу менеджеров | Дашборд «Зависшие брони» с фильтрацией по менеджеру и направлению. Возможность «передать» бронь другому менеджеру |
| US-08 | Как Финансист, я хочу получать реестр броней с финальным неттинг-расчётом для сверки с принимающей стороной | Ежедневный автоматический отчёт (Excel) в 09:00, содержащий: номер брони, поставщик, цена нетто, наценка, статус оплаты |
